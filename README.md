# The Strange Agency

### to run locally:
```
$ git clone git@github.com:thestrangeagency/tsa-website.git
$ cd tsa-website.git
$ bower install
$ npm install
$ gulp serve
```

### to build:
```
$ gulp
```

### to run distribution build:
```
$ gulp serve:dist
```

### to deploy:
```
$ gulp deploy
```
